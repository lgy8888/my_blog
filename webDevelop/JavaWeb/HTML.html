                                          # 基础标签

                                          标签含义
                                          <!DOCTYPE html>html5标识
                                          <meta charset='utf-8'>定义页面的字符集
                                          <h1> - <h6>定义标题（1到6级）
                                          <font>定义文本的字体，尺寸，颜色
                                          <b>定义粗体文本
                                          <i>定义斜体文本
                                          <u>定义文本下划线
                                          <center>定义文本居中
                                          <p>定义段落
                                          <br>定义换行
                                          <hr>定义水平线
                                          • color：颜色的书写方式
                                            • 英文单词：red，blue，pink……
                                            • rgb：rgb(值1，值2，值3)，0~255
                                            • #值1值2值3，00~ff (16进制)

                                          # 多媒体标签

                                          标签描述
                                          <img>定义图片
                                          <audio>定义音频
                                          <video>定义视频
                                          • img：定义图片
                                            • src：规定显示图像的URL
                                            • height：定义图像的高度
                                            • width：定义图像的宽度
                                            • 尺寸单位：px(像素)，%(百分比)
                                          • audio：定义音频（支持的格式：MP3，WAV，OGG）
                                            • src：规定音频的URL
                                            • control：显示播放控件
                                          • video：定义视频（支持的格式：MP4，WebM，OGG）
                                            • src：规定视频的URL
                                            • controls：显示播放控件
                                            • height：定义视频的高度
                                            • width：定义视频的宽度

                                          # 超链接标签

                                          标签描述
                                          <a>定义超链接，用于连接到另一个资源
                                          • href：指定访问资源的URL
                                          • target：指定打开资源的方式
                                            • _self：默认值，在当前页面打开
                                            • _blank：在空白页面打开

                                          # 表格标签

                                          标签描述
                                          <table>定义表格
                                          <tr>定义行
                                          <td>定义单元格
                                          <th>定义表头单元格
                                          • table：定义表格
                                            • border：规定表格边框的宽度
                                            • width：规定表格的宽度
                                            • cellspacing：规定单元格之间的空白
                                          • tr：定义行
                                            • alngn：定义表格行的内容对齐方式
                                          • td：定义单元格
                                            • rowspan：规定单元格可横跨的行数
                                            • colspan：规定单元格可横跨的列数

                                          <span>用于组合行内元素

                                          # 表单标签

                                          • 表单：在网页中主要负责数据采集功能，使用<form>标签定义表单
                                          • 表单项（元素）：不同类型的input元素，下拉列表，文本域等
                                          标签描述
                                          <form>定义表单
                                          <input>定义表单项
                                          <label>为表单项定义标注
                                          <select>定义下拉列表
                                          <option>定义下拉列表的列表项
                                          <textarea>定义文本域

                                          • form：定义表单

                                            • action：规定当提交表单时向何处发送表单数据，URL
                                            • method：规定用于发送表单数据的方式
                                              • get：浏览器会将数据直接附在表单的action的URL之后，大小有限制
                                              • post：浏览器会将数据发送到http请求消息体中，大小无限制

                                          • <input>：表单项，通过type属性控制输入形式，value属性控制提交的值

                                          • <select>：定义下拉列表，<option>定义控制项
                                          • <textarea>：文本域
