                                                      # 链表

                                                      # 基本原理

                                                      • 链表是一种通过指针串联在一起的线性结构

                                                      • 一个节点由数据域和指针域两部分组成，最后一个节点的指针域指向null（空指针）

                                                      # 链表的类型

                                                      • 单链表：一个指针域，指针域指向下一个节点

                                                      • 双链表：两个指针域，一个指向上一个节点，一个指向下一个节点

                                                        • 双链表既可以向前查询也可以向后查询

                                                      • 循环链表：首尾相连的单链表，可以用来解决约瑟夫问题

                                                      # 链表的存储方式

                                                      • 链表是通过指针域的指针链接在内存中的各个节点
                                                      • 链表中的节点在内存中不是连续分布的，是散乱分布在内存的某地址上，分配机制取决于操作系统的内存管理

                                                      # 链表的定义

                                                      public class ListNode {
    // 结点的值
    int val;

    // 下一个结点
    ListNode next;

    // 节点的构造函数(无参)
    public ListNode() {
    }

    // 节点的构造函数(有一个参数)
    public ListNode(int val) {
        this.val = val;
    }

    // 节点的构造函数(有两个参数)
    public ListNode(int val, ListNode next) {
        this.val = val;
        this.next = next;
    }
}

                                                      # 链表的操作

                                                      • 删除节点

                                                        • 将待删除节点的前一个节点的指针指向待删除节点的后一个节点

                                                      • 添加节点

                                                        • 将待添加位置的前一个节点指针指向待添加节点，将待添加节点的指针指向待添加位置的后一个指针

                                                      # 移除链表元素

                                                      力扣题目链接open in new window

                                                      # 基本思路

                                                      两根指针，pre指针和cur指针，前指针用来删除，当前指针用来遍历

                                                      设置虚拟头节点，统一所有节点的移除方式

                                                      pre初始化为虚拟头节点，cur初始化为给定头节点

                                                      while循环遍历整个链表，条件为当前指针不等于空

                                                      循环中判断当前指针的值是否符合条件，符合就删除

                                                      删除后不要忘记更新两根指针

                                                      # 易错点

                                                      • 设置虚拟头指针统一所有节点的移除方式
                                                      • 两根指针，一根用于删除一跟用于遍历
                                                      • 循环遍历链表的条件为当前指针不为空
                                                      • 前指针的更新与否取决于是否有进行移除操作（移除操作本质是更新前指针的next）

                                                      # 代码实现

                                                      class Solution {
    public ListNode removeElements(ListNode head, int val) {
        if(head == null){
            return head;
        }
        // 设置虚拟头指针
        ListNode dummy = new ListNode(-1, head);
        // 初始化前指针为虚拟头指针
        ListNode pre = dummy;
        // 初始化当前指针为给定头指针
        ListNode cur = head;
        // 用当前指针遍历整个链表
        while(cur != null){
            // 判断是否符合条件
            if(cur.val == val){
                // 删除相当于更新了前指针的值
                pre.next = cur.next;
            }else{
                // 没有删除就得更新前指针
                pre = cur;
            }
            // 当前指针无论如何都要更新
            cur = cur.next;
        }
        // 返回
        return dummy.next;
    }
}

                                                      # 设计链表

                                                      力扣题目链接open in new window

                                                      # 基本思路

                                                      • 先定义节点类
                                                        • 属性为当前节点值和下一个节点指针
                                                      • 再定义链表类
                                                        • 属性为头节点和链表长度
                                                      • 涉及到索引的方法都需要判断索引是否大于链表长度是否为负
                                                        • 无效索引根据题目要求返回-1或直接返回
                                                      • 按索引查
                                                        • 判断索引是否合法
                                                        • 定义头节点为当前指针
                                                        • for循环遍历到给定索引处为止
                                                        • 返回当前指针的值
                                                      • 增加
                                                        • 只需要实现按索引增加
                                                        • 加头部则索引为0；加尾部则索引为链表长度size
                                                        • 若索引为负数则改为0，若索引大于链表长度则直接返回
                                                        • 增加需要同时更新链表大小size
                                                        • 定义索引处的前驱节点pre
                                                        • for循环遍历到给定索引处前一个位置
                                                        • 在该位置利用前驱节点pre执行新增操作
                                                      • 按索引删
                                                        • 判断索引是否合法
                                                        • 删除需要同时更新链表大小size
                                                        • 定义索引处的前驱节点pre
                                                        • for循环遍历到给定索引处前一个位置
                                                        • 在该位置利用前驱节点pre执行删除操作

                                                      # 易错点

                                                      • 链表类少个size
                                                      • 忘记判断索引的合法性
                                                      • 合理利用前驱节点来完成增删操作
                                                      • 增加操作允许索引为负数，意思为加在头部
                                                      • 一般用for循环，迭代条件为：指针i < 给定索引index
                                                        • 遍历到给定索引的前驱节点

                                                      # 代码实现

                                                      class MyNode {
    int val;
    MyNode next;

    public MyNode(){}

    public MyNode(int val) {
        this.val = val;
        this.next = null;
    }

    public MyNode(int val, MyNode next) {
        this.val = val;
        this.next = next;
    }
}

class MyLinkedList {
    MyNode head;
    int size;

    public MyLinkedList() {
        size = 0;
        head = new MyNode(0);
    }
    
    public int get(int index) {
        if (index < 0 || index >= size) {
            return -1;
        }
        MyNode cur = head;
        for(int i = 0; i <= index; i++){
            cur = cur.next;
        }
        return cur.val;
    }
    
    public void addAtHead(int val) {
        addAtIndex(0, val);
    }
    
    public void addAtTail(int val) {
        addAtIndex(size, val);
    }
    
    public void addAtIndex(int index, int val) {
        if(index > size) {
            return;
        }
        index = Math.max(0, index);
        size++;
        MyNode pre = head;
        for(int i = 0; i < index; i++){
            pre = pre.next;
        }
        MyNode add = new MyNode(val, pre.next);
        pre.next = add;
    }
    
    public void deleteAtIndex(int index) {
        if (index < 0 || index >= size) {
            return;
        }
        size--;
        MyNode pre = head;
        for(int i = 0; i < index; i++){
            pre = pre.next;
        }
        pre.next = pre.next.next;

    }
}

                                                      # 反转链表

                                                      # 迭代法

                                                      # 基本思路

                                                      一个pre指针，一个cur指针，再多一个tmp指针用来存放cur.next

                                                      循环遍历链表，条件为cur指针不为空

                                                      循环中实现链表倒转逻辑

                                                      # 代码实现

                                                      public ListNode reverseList(ListNode head) {
    // 双指针法
    ListNode tmp = null;
    ListNode pre = null;
    ListNode cur = head;
    while(cur != null){
        tmp = cur.next;
        cur.next = pre;
        pre = cur;
        cur = tmp;
    }
    return pre;
}

                                                      # 递归法

                                                      # 基本思路

                                                      同递归法，只是将循环改为递归

                                                      写个方法传pre指针跟cur指针进去

                                                      设置好边界条件 —— > cur指针为null

                                                      # 代码实现

                                                      public ListNode reverseList(ListNode head) {
    // 递归法
    return reverse(null, head);
}

ListNode reverse(ListNode pre, ListNode cur){
    if(cur == null) {
        return pre;
    }
    ListNode tmp = cur.next;
    cur.next = pre;
    pre = cur;
    cur = tmp;
    return reverse(pre, cur);
}

                                                      # 两两交换链表中的节点

                                                      力扣题目链接open in new window

                                                      # 基本思路

                                                      设置虚拟头指针dummy

                                                      设置ijk三根指针，设置临时变量tmp存放k.next

                                                      迭代时i指向k，k指向j，j指向k.next

                                                      边界条件设为k != null

                                                      # 易错点

                                                      • 返回的是dummy.next
                                                      • 注意head为空和head.next为空的情况
                                                      • 注意迭代过程中出现的两种情况
                                                        • 第二根指针j有可能位于边界，也有可能为null

                                                      # 代码实现

                                                      /**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode swapPairs(ListNode head) {
        if(head == null || head.next == null){
            return head;
        }
        ListNode dummy = new ListNode(-1, head);
        ListNode i = dummy;
        ListNode j = head;
        ListNode k = head.next;
        while(k != null){
            ListNode tmp = k.next;
            i.next = k;
            k.next = j;
            j.next = tmp;

            i = j;
            j = j.next;
            if(j == null) {
                break;
            }else{
                k = j.next;
            }
        }
        return dummy.next;
    }
}

                                                      # 删除链表的倒数第N个节点

                                                      力扣题目链接open in new window

                                                      # 基本思路

                                                      双指针

                                                      创建一个虚拟头指针

                                                      快指针先移动n+1位，与慢指针的距离拉开

                                                      然后快慢指针一起移动，知道快指针碰到链表结尾

                                                      此时慢指针在待移除元素的前一个节点

                                                      完成移除元素操作

                                                      返回虚拟节点的下一个节点

                                                      # 代码实现

                                                      /**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode dummy = new ListNode(-1, head);
        ListNode f = dummy;
        ListNode s = dummy;
        for(int i = 0; i <= n; i++){
            f = f.next;
        }
        while(f != null){
            f = f.next;
            s = s.next;
        }
        s.next = s.next.next;
        return dummy.next;
    }
}

                                                      # 链表相交

                                                      力扣题目链接open in new window

                                                      # 基本思路

                                                      使用hashset存第一个链表

                                                      遍历第二个链表，利用hashset里面的contains方法判断第一个链表中是否存在第二个链表的节点

                                                      存在就返回回去，不存在就返回null

                                                      # 代码实现

                                                      /**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        Set<ListNode> visited = new HashSet<>();
        ListNode tmp = headA;
        while(tmp != null){
            visited.add(tmp);
            tmp = tmp.next;
        }
        tmp = headB;
        while(tmp != null){
            if(visited.contains(tmp)){
                return tmp;
            }
            tmp = tmp.next;
        }
        return null;
    }
}

                                                      # 环形链表

                                                      力扣题目链接open in new window

                                                      # 哈希法

                                                      # 基本思路

                                                      定义一个HashSet集合，用来装遍历过的链表节点

                                                      循环遍历链表，判断之前是否出现过该节点

                                                      出现过则说明进入环了，返回当前节点

                                                      没出现过则说明该链表没有环，返回null

                                                      # 代码实现

                                                      /**
 * Definition for singly-linked list.
 * class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public ListNode detectCycle(ListNode head) {
        Set<ListNode> res = new HashSet<>();
        while(head != null){
            if(res.contains(head)){
                return head;
            }
            res.add(head);
            head = head.next;
        }
        return null;
    }
}

                                                      # 双指针法

                                                      # 基本思路

                                                      定义快慢两个指针fast和slow，快指针一次移动两格，慢指针一次移动一格

                                                      如何判断链表是否有环 ——> 快指针与慢指针是否相遇

                                                      如何找到环的起点：

                                                      将链表起点到环起点的距离设为 $x$，将环起点到快慢相遇点的距离设为 $y$，快慢相遇点到环起点的距离设为 $z$

                                                      如图易得等式： $slow=x+y$ 和 $fast=x+y+n(y+z) ①$

                                                      由快慢指针的速度可得 ：$2slow = fast②$

                                                      结合①②式子可得：$2(x+y)=x+y+n(y+z)$

                                                      解得：$x=n(y+z)-y$

                                                      将$n(y+z)$拆一个出来可得：$x=(n-1)(y+z)+z③$

                                                      由③可得：链表起点到环起点的距离等于$(n-1)$圈环的长度加上快慢指针相遇点到环起点的位置

                                                      设$n=1$，易得：$x=z$

                                                      所以，只要在链表起点处设置指针index1，在快慢指针相遇处设置指针index2

                                                      让这两个指针移动相同的距离，相遇点就是环的起点

                                                      # 易错点

                                                      • 快指针至少转了一圈才能跟慢指针相遇，说明$n>=1$
                                                      • 慢指针无法在环中转够一圈，因为慢指针转1圈的时间快指针可以转2圈

                                                      # 代码实现

                                                      /**
 * Definition for singly-linked list.
 * class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode fast = head;
        ListNode slow = head;
        while(fast != null && fast.next != null){
            fast = fast.next.next;
            slow = slow.next;
            if(fast == slow){
                ListNode index1 = head;
                ListNode index2 = fast;
                while(index1 != index2){
                    index1 = index1.next;
                    index2 = index2.next;
                }
                return index1;
            }
        }
        return null;
    }
}
